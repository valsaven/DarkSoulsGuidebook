# Dark Souls Guidebook

## Project Setup

```sh
npm i
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint Files

```sh
npm run lint
```

### Lint and Fix Files

```sh
npm run lint:fix
```
